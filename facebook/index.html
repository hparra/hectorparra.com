The "Like" Button

See: Like Button

See: JavaScript SDK Reference

The "Like" button can be a tricky deal, because you may only Like domains that are registered with Facebook. This is already done if you create a Page or App. If not, you must register the domain with Insights. You may have to edit your website to add the fb:admin code.

Facebook announced it is changing how the "Like" button works on November 7, 2012. See "Like Button Migration" for details.
